Automatically creates links in your content based on WordPress tags and custom words.

== Description ==
Auto Tag Linker automatically converts words in your content into links based on your WordPress tags and a custom word list.

Features:
* Link words based on WordPress tags
* Custom word list with specific URLs
* Blacklist to exclude words
* Custom CSS styling
* Control per post
* Multiple post type support

== Installation ==
1. Upload the plugin files to wp-content/plugins/auto-tag-linker
2. Activate the plugin
3. Go to Settings > Auto Tag Linker to configure

== Frequently Asked Questions ==
= Can I customize the link style? =
Yes, you can add custom CSS in the plugin settings.

= Can I prevent specific words from becoming links? =
Yes, use the blacklist feature in settings.
